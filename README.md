# glowy-random-boolean

> Generate a random boolean

## Install

```
$ npm install --save glowy-random-boolean
```

## Usage

```js
const randomBoolean = require('glowy-random-boolean');

randomBoolean();
//=> true

randomBoolean();
//=> false
```

## Tests

Run tests

```
npm run test
```
