# random-boolean

> Generate a random boolean

## Install

```
$ npm install --save @glowyjs/random-boolean
```

## Usage

```js
const randomBoolean = require('@glowyjs/random-boolean');

randomBoolean();
//=> true

randomBoolean();
//=> false
```

## Tests

Run tests

```
npm run test
```
